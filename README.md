# Nice-KT
基于Kotlin + Vert.x 分布式服务端框架



服务器架构 ：

* 登录服：登录验证
* 网关服：所有消息转发
* 游戏服：游戏逻辑服
* 战斗服：战斗逻辑服

## 运行环境

JDK1.8以上， Idea2020.2 最新版经常有右键失灵的Bug

单机运行执行： AppAllInOne.kt
